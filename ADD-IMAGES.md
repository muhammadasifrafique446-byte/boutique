## 🐛 Troubleshooting

### Images Not Showing?

**Check 1: File Names**
- Open GitHub → boutique repo → images folder
- Verify files are named: `design1.jpg`, `design2.jpg`, etc.
- NO spaces, NO extra characters

**Check 2: Folder Structure**
- Images must be in: `images/design1.jpg`
- NOT in: `design1.jpg` (at root level)

**Check 3: Wait Time**
- After uploading, wait 2-3 minutes for deployment
- Refresh page: Press Ctrl+F5 (or Cmd+Shift+R on Mac)

**Check 4: GitHub Pages Settings**
- Go to: GitHub → Settings → Pages
- Verify: Source = `main` branch
- Verify: Status shows "Your site is live at..."

### WhatsApp Buttons Not Working?
